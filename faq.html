SYSTEM GUIDE

BookMySalon system kaise kaam karta hai?

Customer side

Nearby salon dhundo ya QR scan karo, service select karo, slot choose karo, aur booking confirm karo.

Salon owner side

Salon register karo, location save karo, booking open karo, staff count set karo, aur live slots manage karo.

Customer ko kaunsi samasya aati hai?

  • Nearby salon list nahi aa rahi: Location permission allow karo, phir page reload karo.
  • Slot full dikh raha hai: Dusra time ya dusra salon select karo.
  • Booking confirm nahi ho rahi: Network check karo, phir 30 sec baad retry karo.
  • WhatsApp chat open nahi ho rahi: App update karo ya WhatsApp Web try karo.

Salon owner ko kaunsi samasya aati hai?

  • Registration complete nahi ho raha: Name, city, state, WhatsApp number sahi format me bharo.
  • Nearby me salon rank low hai: Dashboard me exact location update karo.
  • Customers ko slot nahi mil raha: Karigar count aur opening/closing time sahi set karo.
  • Booking inactive dikh raha: Subscription status check karo.

Registration ka step-by-step

  1. Register page open karo aur salon + owner details bharo.
  2. Location permission do ya address manually fill karo.
  3. WhatsApp number verify karo.
  4. Salon ID aur booking link save karo.
  5. Dashboard me booking open karke slots activate karo.

Find Nearby kaise use kare?

  1. Find Nearby page open karo.
  2. Location allow karo.
  3. Radius select karo (3km, 5km, 10km).
  4. Search box me salon/city type karo.
  5. Book Now par click karke slot choose karo.

Beauty parlour ke liye dynamic slot (recommended)

Haan, beauty parlour add karte waqt dynamic slot best rahega.

  • Har service ki duration set karo (jaise threading 15 min, facial 60 min).
  • Staff-wise availability maintain karo.
  • Overlap-free slot generation rakho.
  • Multi-service booking me combined duration apply karo.

Beauty Parlour FAQ

Q: Beauty parlour registration alag hai kya?

A: Nahi, same registration page me services checkbox se beauty parlour services select karo.

Q: Threading/Facial ke slots kaise set honge?

A: Duration-based dynamic slot matching hota hai, isliye 15 min aur 60 min services dono ke liye alag valid start times milte hain.

Q: Bridal ya long service ka booking kaise hota hai?

A: System continuous multiple slots reserve karta hai, isliye clash kam hota hai.
